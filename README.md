1. Title/Headline
   
    blinkit_Dashboard : A comprehensive view of sales, customers, and product performance.
 
 3. Description
    
    Developed a comprehensive Blinkit Grocery Sales Dashboard in Power BI to track and analyze key performance indicators, including 
    total sales, average sales, customer ratings, outlet performance, and product category trends. Utilized DAX measures, data modeling, 
    and interactive visualizations to deliver actionable insights that support business decision-making and performance optimization.

4. Tech Skills
   
   Power Query (ETL):
   
          Imported and transformed raw Blinkit sales data.
   
          Cleaned and standardized data by handling missing values, duplicates, and inconsistent formats.
   
          Created custom columns and transformed data to support business analysis.
   
          Automated data preparation workflows.

   DAX (Data Analysis Expressions) : Developed calculated measures and KPIs, including:

           Total Sales,Average Sales,Number of Items Sold
           Average Rating,Item Visibility,Used aggregation functions,
           filters, and context-based calculations for dynamic reporting

   Data Visualization 
   Implemented various visual elements: KPI Cards,  
                                        Donut Charts,
                                        Bar Charts   
                                        Area Chart   
                                        Tables and Matrix Visuals   
                                        Interactive Slicers   
                                        Custom Icons and Navigation Buttons

       
6. Data Source

         Microsoft Excel :Used as the primary data source.
         Performed initial data validation and preprocessing before loading into Power BI.

7. Business Intelligence & Analytics
     
    Sales Performance Analysis
   
    Product Category Analysis
   
    Outlet Performance Monitoring
   
    Customer Rating Analysis
   
    Inventory and Product Visibility Tracking
   
    Trend Analysis and Decision Support
